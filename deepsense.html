Pipeline

How It Works

A five-stage intelligent pipeline transforms raw underwater audio into actionable environmental intelligence.

🎵
1
Audio Input
WAV, MP3, or FLAC file uploaded via the desktop GUI. Sampled at 22,050 Hz.
📊
2
Feature Extraction
Mel Spectrogram (128×215) and 42-dimensional MFCC features extracted using Librosa.
🧠
3
Multi-Model Inference
CNN + MFCC ensemble predicts species, category, biotic class and distance.
🤖
4
Transformer Verification
Marine1 HuggingFace model cross-validates category with confidence score.
⚠️
5
Hazard Classification
Rule engine combines all outputs to flag events as Hazardous or Safe.

Capabilities

What DeepSense Detects

Six intelligent outputs from a single audio file — no human expertise required.

🐬
Species Identification
Identifies marine species including Dolphins, Humpback Whales, Orcas, Seals and Sea Lions from their unique acoustic signatures.
CNN · 88%+ confidence
🏷️
Category Classification
Classifies sounds into Marine Animals, Vessels, Natural Sounds, or Human Sounds — providing environmental context.
MFCC Classifier
🌿
Biotic / Abiotic Detection
Determines whether the sound originates from a living organism (biotic) or a non-living mechanical source (abiotic).
Binary Classifier
📡
Distance Estimation
Estimates the distance of the sound source in metres using a log-transformed regression model trained on MFCC features.
Regression Model
⚠️
Hazard Detection
Automatically flags Explosions, Sonar, and Vessel activity as environmental hazards using a rule-based classification engine.
Rule Engine
🖥️
Offline Desktop App
Full PyQt5 dashboard with drag-and-drop analysis, persistent history, analytics charts — runs 100% offline with no internet.
PyQt5 · Matplotlib

Architecture

The Model Pipeline

Five specialized models work together in an ensemble — each handling a distinct prediction task.

Species CNN
Deep Learning · .h5
Conv2D network trained on 128×215 mel spectrograms. Outputs softmax probability over all species classes.
88%
MFCC Species
Classical ML · .pkl
Fallback classifier using 42-dim MFCC vectors. Activated when CNN confidence falls below 60%.
78%
Category Model
Classical ML · .pkl
Classifies audio into Marine Animals, Vessels, Natural Sounds or Human Sounds from MFCC features.
91%
Biotic Classifier
Classical ML · .pkl
Binary classifier distinguishing living organism sounds from mechanical or environmental noise.
93%
Distance Regressor
Regression · .pkl
Log-transformed regression estimating source distance in metres from MFCC features using expm1 inverse.
R²: 0.74
Marine1 Transformer
HuggingFace · .safetensors
Fine-tuned transformer model providing independent high-level category verification with confidence score.
97%

About the Project

Built for Real-World
Marine Monitoring

DeepSense was developed as a B.E Final Year Project to address the challenge of automated, offline underwater environment surveillance using passive acoustic monitoring and AI.

🎓

Academic Context

B.E / B.Tech Final Year Project — Computer Science / AI & ML stream. Developed and presented to external jury.

🌍

Real-World Applications

Marine conservation, naval defense, port security, environmental impact assessment of shipping routes.

🔮

Future Scope

Real-time hydrophone input, GPS tagging, larger dataset with augmentation, web-based remote dashboard.
